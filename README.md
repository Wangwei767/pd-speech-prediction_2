### 📖 系统简介
本系统是基于机器学习的帕金森病语言障碍临床预测工具，为医生提供科学的决策支持。

### 🔬 学术用途
本工具专为学术研究和临床辅助诊断设计，请在专业医生指导下使用。

### 📄 使用说明
1. 输入患者临床数据
2. 系统自动计算风险评分
3. 查看预测结果和解释
4. 导出报告供临床参考
